TextPad+ — это бесплатный текстовый редактор с открытым исходным кодом, предназначенный для быстрого создания и редактирования текстовых документов. Для новых версий программы необходима установленная для Windows библиотека .NET 6 Runtime Desktop. Для версий с 1.4 по 1.8 включительно нужны библиотеки .NET Core 3.1 и .NET 5 Runtime Desktop. Для версий старее нужна только библиотека .Net Core 3.1 Runtime Desktop. Работает на архитектуре x86 и поддерживает Windows 7.
