TextPad+ — это текстовый редактор с открытым исходным кодом, способный быстро создавать простые текстовые файлы. Работает на ПК архитектуры x86 и ОС Windows 7 и выше.

Инструкци по установке:
Программа, начиная с версии 2.5.0, доступна для скачивания с ветки релиза на GitHub и не требует установки дополнительных компонентов.
Для версий с 1.9 по 2.4.0 включительно нужно установить для библиотеку .NET 6.0 Desktop Runtime. Для версий с 1.4 по 1.8 включительно нужны библиотеки .NET Core 3.1 и .NET 5 Runtime Desktop. Для более старых версий нужно установить только библиотеку для .NET Core 3.1.
