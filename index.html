TextPad+ — это бесплатный текстовый редактор, предназначенный для быстрого создания и редактирования текстовых документов. Для новых версий программы необходима установленная для Windows библиотека .NET Runtime Desktop 6.0. Для версий с 1.4 по 1.7 включительно нужны библиотеки .NET Runtime Desktop 5 и .NET Core Runtime Desktop 3.1 . Для версий старее нужна только библиотека .Net Core Runtime Desktop 3.1. Работает на архитектуре x86.
