TextPad+ — это бесплатный текстовый редактор, предназначенный для быстрого создания и редактирования текстовых документов. Для новых версий программы необходима устаовленния для Windows библиотека .NET SDK 6.0. Для версий с 1.4 по 1.7 включительно нужны библиотеки .NET SDK 5 и .NET Core SDK 3.1 . Для версий старее нужна только библиотека .Net Core SDK 3.1. Поддерживаются процессоры на архитектуре x86.
.Net 6.0: https://dotnet.microsoft.com/en-us/download/dotnet/6.0
.Net 5.0: https://dotnet.microsoft.com/en-us/download/dotnet/5.0
.Net 3.1: https://dotnet.microsoft.com/en-us/download/dotnet/3.1
