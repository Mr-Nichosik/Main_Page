Ключевые особенности: наличие вкладок, возможна работа с несколькими файлами параллельно; английская локализация; дополнительные инструменты для запуска некоторых форматов файлов; не требует установки дополнительных компонентов. При запуске установщика программы, Windows может предупредить о том, что программа небезопасна. Вы можете проигнорировать это, так как весь исходный код программы доступен на GitHub.
